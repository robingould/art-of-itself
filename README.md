# Art of Itself
Ascii art of a word in the shape of itself!

# How to use :0
``` bash
go build art-of-itself.go
./art-of-itself <desired word>
```

# Example
``` bash
./art-of-itelf what
          wh                    at      
wh  at    whatwh      atwh    atwhat    
whatwh    at  wh    at  wh      at      
whatwh    at  wh    atwhat      what
```
